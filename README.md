# 📊 FUTURE_DS_02
Customer Churn Analysis (Excel)

## 📌 Project Overview
This project analyzes customer churn data to identify patterns, retention drivers, and business insights.

## 🛠 Tools Used
- Excel (Pivot Tables, Charts, Dashboard)

## 📊 Key Analysis
- Churn by Contract
- Churn by Tenure
- Churn by Payment Method
- Churn by Monthly Charges
- Retention Analysis

## 📈 Key Insights
- Month-to-month customers churn more
- New customers are at higher risk
- High charges increase churn
- Electronic check users churn more

## 💡 Recommendations
- Promote long-term contracts
- Improve onboarding
- Encourage auto-payments
- Focus on high-risk customers
